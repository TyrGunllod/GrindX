Design Detalhado - Card Inline

Estrutura e comportamento do card expandido

Exemplo: Módulo expandido
gestao-vendas v2.1.0
Importado schema: public
MÓDULO
gestao-vendas
VERSÃO
2.1.0
SCHEMA
public
estoque v1.0.0
Novo schema: private

Comportamento

  • Clique na linha: Expande/recolhe o card
  • Ícone de seta: Indica estado (▼ expandido, ► recolhido)
  • Múltiplos expandidos: Apenas um por vez (accordion)
  • Animação: Transição suave de 200ms

Conteúdo do Card

  • Informações básicas: Nome, versão, schema (em grid 3 colunas)
  • Botão de ação: Importar (azul) ou Reimportar (amarelo)
  • Botão secundário: Detalhes (para futuras expansões)

Acessibilidade

  • Teclado: Tab para navegar, Enter/Space para expandir
  • Screen reader: aria-expanded, aria-controls
  • Touch: Área de clique mínima 44px
