       )

       ```python
       op.create_index(
           "ix_usuarios_role",
           "usuarios",
           ["role"],
           unique=False,
           schema="iam",
       )
       ```

    3. **usuarios ativo index** — for `listar_ativos` query pattern:
       ```python
       op.create_index(
           "ix_usuarios_ativo",
           "usuarios",
           ["ativo"],
           unique=False,
           schema="iam",
       )
       ```

    4. **usuarios empresa_id index** — for joins with empresas:
       ```python
       op.create_index(
           "ix_usuarios_empresa_id",
           "usuarios",
           ["empresa_id"],
           unique=False,
           schema="iam",
       )
       ```

    5. **portal_modulos aba_id index** — for module lookups by tab:
       ```python
       op.create_index(
           "ix_portal_modulos_aba_id",
           "portal_modulos",
           ["aba_id"],
           unique=False,
           schema="portal",
       )
       ```

    Downgrade: drop all indexes in reverse order using `op.drop_index(name, table_name=table, schema=schema)`.
