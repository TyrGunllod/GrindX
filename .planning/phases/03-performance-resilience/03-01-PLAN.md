    2. Create `apps/api-postgres/app/core/cache.py` with:
       - Import `TTLCache` from `cachetools` and `threading`
       - Three cache instances: `_theme_cache = TTLCache(maxsize=100, ttl=900)`, `_portal_cache = TTLCache(maxsize=100, ttl=900)`, `_user_cache = TTLCache(maxsize=100, ttl=900)`

       - Helper function `clear_all()` that clears all three caches (for testing)
